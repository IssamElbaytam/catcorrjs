catcorr.js

Visualize correlations across many dimensions of categorical data

Download this project as a .zip file Download this project as a tar.gz file

These data were generated from a nonsensical survey at the announcement of catcorr.js. The data are live-updated from a google spreadsheet in case you, too, would like to play with the data.

To use catcorr.js, brush any dimension to reveal how the selected people answered other questions (darker bars) but without changing how everyone responded to the survey (lighter bars). As a means of comparison, the white lines indicate how the selected group would have answered the remaining questions if their answers were completely uncorrelated.
